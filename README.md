Go 语法基础示例集合

每个示例在独立目录下，可以单独运行：

- 01_hello
- 02_variables
- 03_control
- 04_functions
- 05_arrays_slices
- 06_maps
- 07_structs
- 08_keywords
- 09_loops
 - 10_pointers
 - 11_errors
 - 12_io
 - 13_packages
 - 14_testing
 - 15_concurrency

运行示例（在仓库根目录）：

```bash
# 运行 Hello 示例
go run ./examples/01_hello

# 运行变量示例
go run ./examples/02_variables

# 运行控制流示例
go run ./examples/03_control

# 运行函数示例
go run ./examples/04_functions

# 运行数组与切片示例
go run ./examples/05_arrays_slices

# 运行映射示例
go run ./examples/06_maps

# 运行结构体示例
go run ./examples/07_structs

# 运行关键字示例
go run ./examples/08_keywords

# 运行更详细的循环示例
go run ./examples/09_loops

# 运行指针示例
go run ./examples/10_pointers

# 运行错误处理示例
go run ./examples/11_errors

# 运行文件 I/O 示例
go run ./examples/12_io

# 运行包示例
go run ./examples/13_packages

# 运行并发示例
go run ./examples/15_concurrency

# 运行测试
go test ./examples/14_testing
```
